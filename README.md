# Precision Smile Orthodontics

Marketing site built with Vite, React, TypeScript, Tailwind, and shadcn-ui.

## Local development
1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`
3. Build for production: `npm run build`

## Project structure
- `src/` – React app code and styling
- `public/` – static assets
- `index.html` – entry HTML with SEO meta tags

## Additional notes
- Configure environment variables (e.g., API endpoints) in `.env`.
- Lint the project with `npm run lint`.
